About this project

This project developed out of the Stanford University data journalism class COMM 177T: Building News Applications in Spring 2021. Many thanks to Professor Serdar Tumogoren for his support. As mentioned on the homepage, all data for this website comes from the EPA's Superfund Program. This website and the underlying calculations were made using python, jinja2, vegalite, flourish, html & css. All of the code for this project can be found here.
