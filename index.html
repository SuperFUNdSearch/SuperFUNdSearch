A note about methodology

All data used for this project is from the EPA's Superfund Program. Data was scraped from the EPA website on May 23rd, 2021 and at the time of publishing this website, May IDK, 2021, total number of days on list calculation is up to date and all other info is from the 23rd scrape. While the milestone timeline visualization on each site page displays each milestone as-is from EPA data, in the caluclations done for the stats and graphs, any milestone that was out of chronological order was dropped. This did not have a large effect on overall themes or calculations. See the about page for more information about this website.
